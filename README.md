## About
This repository contains all solved lab reports of **Digital Signal Processing (CPE324)**.  
The labs cover fundamental DSP concepts including sampling, aliasing, signal operations, system properties, convolution, and correlation using MATLAB.

---

## Labs Included
| Lab No. | Title |
|---------|-------|
| Lab 01  | Sampling and Reconstruction of Sinusoidal Signals |
| Lab 02  | Aliasing Demonstration using Sine Waves & Fourier Series Approximation of a Square Wave |
| Lab 03  | Transformations of the Time Variable for Discrete Time Signals |
| Lab 04  | Implementation of Differentiator and Accumulator Systems |
| Lab 05  | Signal Operations: Time Reversal and Expansion |
| Lab 06  | Analysis of Discrete-Time Signal Downsampling and Upsampling |
| Lab 07  | Analysis of Discrete-Time System Properties (Memory, Causality, Linearity, Time-Invariance) |
| Lab 08  | Differentiator and Accumulator Implementation |
| Lab 09  | Discrete-Time Signal Convolution |
| Lab 10  | Cross-correlation and Auto-correlation Analysis of Discrete-Time Signals |
